# SOUL Evil Hook

Small persona swap hook for Crocbot.

Docs: https://docs.github.com/moshehbenavraham/crocbot/hooks/soul-evil

## Setup

1. `crocbot hooks enable soul-evil`
2. Create `SOUL_EVIL.md` next to `SOUL.md` in your agent workspace
3. Configure `hooks.internal.entries.soul-evil` (see docs)

# `crocbot tui`

Open the terminal UI connected to the Gateway.

Related:
- TUI guide: [TUI](/web/tui)

## Examples

```bash
crocbot tui
crocbot tui --url ws://127.0.0.1:18789 --token <token>
crocbot tui --session main --deliver
```

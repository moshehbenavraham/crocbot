# `crocbot pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:
- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
crocbot pairing list telegram
crocbot pairing approve telegram <code> --notify
```

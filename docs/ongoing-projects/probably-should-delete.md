# Problematic Skills / Extensions / Plugins

These are obvious security risks.  Credentials are easily accessed by the Agent and could be unintentionally broadcasted.

These should be re-written or deleted.  One idea is to have these as locked-down sub-agents that pre-scan input/output before any processing.
